# Project Name xd

## Run Locally
```
python -m venv venv
\venv\Scripts\activate
pip install -r requirements.txt
python main.py
````

## Build

Activate venv and install PyInstaller
Then build with: `python -m PyInstaller .\main.py --onefile --noconsole --add-data "languages.json;."`
